
# Analog Clock

### *Features*:
* User-friendly
* Displays current Time
* Neomorphic Interface
* Added Themes: Dark Mode <-> Light Mode


## *Features to be added*

- Adding Timezones
- Adding a stopwatch / timer (countdowns)


## *Technologies used*

- HTML5
- CSS3 (with Flexbox)
- Vanilla Javascript
